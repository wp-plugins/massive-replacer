Massive Replacer lets you replace a string determined by a different.

== Description ==

With Massive Replacer you can replace strings of your posts by other different.
You can, among other things, put in bold text of all articles or replace your letter combinations for various other.

The strings that you want replace are placed in the WordPress database.


== Usage ==

Go to Massive Replacer option in the WordPress admin panel. Next you will see an a form with 2 textarea.
In the first textarea you must write the strings to replace. You can do meny replacement. Write "]]||[[" between each string and the plugin will think
that they're different strings.

In the second textarea you must write the strings that you want to appear. You can do the same that in the first textarea.


== Installation ==

1. Unzip `massive_replacer.zip`
2. Upload `massive_replacer.php` to the 'wp-content/plugins' directory
3. Activate the plugin through the 'Plugins' menu in WordPress
4. Enjoy :) !
